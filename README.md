# PINBALL-WIZARD-26-GAMEPAD
This is the home of the great flexible Pinball Controller Pinball Wizard 26 (Work in progress)  

Live Preview work in progress allmost done. (08.04.2026)  
[![Display Modell](PBWZ26RGB-7SEG-P4-43Inch_display.JPEG)](https://a360.co/4cjfFVw)

This is a link to a fusion360 server, you can preview the new model in your browser.  

Software developement for the new MCU has NOT begun. It start next days, after printing.  
Expected Printing time 5 days, with breaks.     

The new hardware platform is the ESP32 P4 + ESP32 C8 for modern bluetooth/wifi communication.    
The middle Part is modular. If you need, you can insert your own Display/Mcu unit.    

New features:  
Display 3.2 --> 4.3 Inch  
change from 320x240 pixel display to 800x480px  
Sound in / output  
Bluetooth x.x --> 5.x (2 times faster in theory. and its an independent chip only for BT)  
cpu xtensa 5 (200mhz dual core) to risc (360mhz dual core)    
cpu with many modern interfaces and many accelerators  
12000mAh powerbank with display included for the RGB leds and charge devices  
extendable 10 cm --> 14 cm. (equals 50 - 60 --> 50 - 64 cm width)  
30mm + 24mm button next on a flexible rotor for best ergonomics  
rgb lights adressable all over  
no soldering on the pcb, only wires. much easier for not skilled builder.  
changes screws to allmost all M3 10mm threat. (over 50 scres in total...)  
bigger rotor size  
optional magnetic feed for cloth drying racks  
foot pedal for full body controll (wireless)  
much improved cableways for the much more required cables now.  
easier to build alu rods. tools to mark and drill it is included.  
release date for the software is maybe xmas. no pressure.
